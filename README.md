# XENO Cards VaultJam2025
XENO Cards - Vault Jam 2025
Xeno Cards is a strategic card game developed with Unity in December 2025 for the Vault Jam 2025. This project was created as part of a time-limited (48 hours) game jam, focusing on innovative gameplay mechanics and visual style.

# Play the Game
You can download the latest build of Xeno Cards on itch.io: [Click here to play on itch.io](https://mehmetkerimartan.itch.io/xeno-cards)

# About the Project
Xeno Cards blends tactical card combat with a distinct atmosphere. It was designed and prototyped within the constraints of Vault Jam 2025, showcasing the synergy between technical development and game art.
